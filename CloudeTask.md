Описание проекта для Claude Code

Мы разрабатываем голосового и текстового ассистента для Windows, концептуально похожего на JARVIS.
Проект разделён на несколько модулей, и каждый участник команды отвечает за свою область.

Команда проекта

Voldemar — разработка ядра на C#.

Yorik — модуль ИИ и Python‑часть.

Taldera — обучение и выполнение базовых задач на C#.

Claude Code — помощь в разработке ядра C# в папке /core.

Папки проекта и правила работы

/core — ядро на C#.
Здесь работает Claude Code. Она может изменять, создавать и удалять файлы только внутри этой папки.

/ai-python — Python‑бэкенд ИИ.
Claude Code может только смотреть, но не вносить изменения.

/ui — будущий интерфейс.
Просмотр разрешён, изменения запрещены.

/docs — документация.
Просмотр разрешён, изменения запрещены.

Claude Code может анализировать файлы во всех папках, чтобы понимать контекст проекта, но изменять код можно только в /core.

Назначение C# модуля

C#‑часть проекта — это ядро ассистента, выполняющее системные действия Windows и обеспечивающее взаимодействие между ИИ (Python) и операционной системой.

Основные задачи C#:

Принимать команды от Python через HTTP API.

Выполнять системные действия: открытие приложений, работа с файлами и папками.

Возвращать структурированный JSON ответ.

Реализовать безопасное выполнение команд.

Формат JSON команд

Входящая команда содержит:

action — тип выполняемого действия

params — параметры операции

uuid — уникальный ID запроса

timestamp — время запроса

Формат ответа

Ответ от C# должен включать:

status — ok или error

result — данные результата

error — описание ошибки, если есть

HTTP API

POST /action/execute — выполнение команды

GET /system/status — проверка статуса
