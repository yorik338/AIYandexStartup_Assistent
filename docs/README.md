# AIYandexStartup_Assistent

Создаём персонального ИИ-ассистента для Windows, который работает на уровне операционной системы, выполняет команды голосом/текcтом, управляет файлами, приложениями, настройками ПК, умеет автоматизировать сложные процессы и безопасно подключается к локальным данным пользователя.

## Модули Python

Папка `ai-python/ai_assistant` содержит основные заготовки:

- **prompts.py** — системный промпт и сборка финального запроса для LLM.
- **llm.py** — отправка промптов, унификация ошибок и вспомогательный echo-бекенд для разработки.
- **schemas.py** — минимальная схема JSON-команд с валидацией обязательных полей.
- **nlu.py** — преобразование текста → JSON: вызов LLM и проверка структуры.
- **speech.py** — заглушки для транскрибации аудио в текст (файл или поток).
- **bridge.py** — отправка команд в C# через HTTP в формате JSON.
- **pipeline.py** — end-to-end пайплайн: текст/аудио → интент → JSON → действие.

## Пример запуска

Файл `ai-python/main.py` демонстрирует последовательные вызовы пайплайна с текстом, аудиофайлом и потоковым аудио. Перед запуском настройте адрес C#-эндпойнта (по умолчанию `http://localhost:5055`).

## Настройка окружения

Сервис использует ChatGPT для распознавания команд и транскрибации аудио. В корне репозитория расположен `.env` с ключевыми переменными:

- `OPENAI_API_KEY` — API-ключ OpenAI (обязателен).
- `OPENAI_API_BASE` — необязательная кастомная база для Azure/OpenAI-совместимых эндпойнтов.
- `OPENAI_MODEL` — модель для чат-комплитов (по умолчанию `gpt-4o-mini`).
- `OPENAI_TRANSCRIPTION_MODEL` — модель для транскрибации аудио (по умолчанию `whisper-1`).
