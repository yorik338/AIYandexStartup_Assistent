# AIYandexStartup_Assistent
Создаём персонального ИИ-ассистента для Windows, который работает на уровне операционной системы, выполняет команды голосом/текстом, управляет файлами, приложениями, настройками ПК, умеет автоматизировать сложные процессы и безопасно подключается к локальным данным пользователя.
